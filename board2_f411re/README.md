# Board2 F411RE Base Joint CAN Test

최종 통합 프로토콜 기준 Board2 테스트 코드입니다.

- Board2 대상: 베이스 1축 / `base_joint`
- 서버 Global Joint ID: `0`
- Board2 Local Motor ID: `0`
- Move CAN ID: `0x102`
- Status CAN ID: `0x202`
- Position Feedback CAN ID: `0x302`
- MCP2515 CS: `PB12`
- MCP2515 INT: `PB4`

자세한 테스트 방법은 `README_BOARD2_F411RE_BASE_FINAL_TEST.md`와 `Board2_CAN_Protocol_FINAL_base_0x302.md`를 확인하세요.
