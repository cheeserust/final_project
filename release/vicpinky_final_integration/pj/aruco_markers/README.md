검출 설정은 `DICT_4X4_100`, marker black-square 외곽 크기 50 mm다. PNG 자체에는
quiet zone이 없으므로 출력할 때 사방에 최소 한 셀 폭 이상의 순백색 여백을 둔다.
프린터의 “페이지에 맞춤”을 끄고 실제 외곽 크기를 자로 확인한다.
